# Jiale Wang - Resume Website

A clean, responsive single-page resume built with plain HTML and CSS.

## Highlights

- Fixed sidebar + scrollable content layout
- Mobile-friendly responsive design
- Smooth section navigation
- Lightweight, no build tooling required

## Tech Stack

- HTML5
- CSS3 (custom properties, responsive layout)

## Quick Start

Run a local static server:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

Alternative:

```bash
npx serve .
```

## Project Structure

```text
.
├─ index.html    # Main page (content + styles)
└─ avatar.jpg    # Profile image
```

## Notes

- This repository is intentionally simple: no bundler, no framework, no test runner.
- To customize content, edit sections directly in `index.html`.
